# Overview

Venomous Villain (VV) is a simple Snake game developed using the Pygame framework. The game uses built-in Pygame functionality to handle object movement, generate food items, play sound effects, and detect collisions between game elements. Developing this project provided insight into how Python can be used to create interactive games that are not graphically intensive while introducing core concepts of game development.

The objective of the game is to control the snake using the keyboard arrow keys and guide it toward food items that appear on the screen. Each time the snake eats food, its length increases and the player’s score rises. The game ends if the snake collides with the boundaries of the game window or with its own body.

{Describe your purpose for writing this software.}
I developed this software to gain hands-on experience with the Pygame library and to strengthen my understanding of fundamental game development concepts. Through this project, I learned how to work with game surfaces, implement a game loop, process user input through events, manage object movement, and detect collisions. The Snake game provided an approachable way to explore these concepts without becoming overwhelmed by the complexity of larger game-development projects.

# Development Environment

{Describe the tools that you used to develop the software}
* Visual Studio Code (VS Code) as the primary development environment
* Mac Terminal for running and testing the application

{Describe the programming language that you used and any libraries.}
The game was developed using Python 3.14. The primary library used was Pygame, which provided the tools necessary for graphics rendering, event handling, collision detection, and audio playback. Additionally, the built-in random module was used to generate food locations at random positions on the game board. The pygame.mixer module was used to incorporate background music and gameplay sound effects.
